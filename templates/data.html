Data

The dataset we used comes from Kaggle. It contains about 130,000 rows of wine reviews from Wine Enthusiast. We used machine learning to try and predict the wine varietal using words from the review/description.

Column Data

Column Description
description sommelier description of the wine's taste
designation vineyard within winery
points rating by Wine Enthusiast
price cost of one bottle of wine
province Yoshi Tannamuri
region_1 wine growing area
region_2 specific region
taster_name wine taster
taster_twitter_handle twitter account of wine taster
title title of wine review (often contains vintage)
variety type of grapes
winery winery that made the wine
